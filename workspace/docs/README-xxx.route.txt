Explanation of .route files:

CHANY -> Vertical track of the connection box
CHANX -> Horizontal tracks of the connection box
Node -> Number of associated to each node
Track -> Number associated to each track (in addition to its node number)
Switch -> Switch type (I think) with 1=switch over a single wire, 2=switch covers multiple line track of the connection box
Pad -> Connection to the I/O
Pin -> Connection to the CLB
